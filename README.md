                # MySplits 📱

A simple and efficient way to manage splits.

## 🚀 Features
* Clean Material Design UI.
* Built with Kotlin.
* export expenses in pdf.
* No ads.Free to use
* Multiple groups & multiple members can be added.
* Weighted splitting possible.
* Day Night theme.
